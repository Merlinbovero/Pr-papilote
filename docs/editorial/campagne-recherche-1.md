## 3. Matrice des opportunités (synthèse priorisée)

Priorité 1 — ALAT (sources : ALAT.fr B + defense.gouv.fr A + Opex360 B) :
organisation actuelle (COMALAT/4e BAC), EALAT et cursus pilote (Dax → Cannet, VOLTAC),

≈ 10-12 fiches nouvelles + enrichissement des 5 existantes + termes + quiz.

Priorité 2 — EOPN/AAE (sources : devenir-aviateur A + Traditions-Air B + Escadrilles B) :
cursus pilote AAE (fiche formation), EAC 00.315 « Christian Martell » (fusion 2020), PC-21,

escadrons emblématiques. ≈ 10-12 fiches.

Priorité 3 — lacunes EOPAN (ARDHAN B + Persée A) : flottilles manquantes (21F, 23F, 24F,
31F-36F par lots), anciens appareils (Super-Étendard Modernisé, Alizé, Crusader, Étendard IV),
almanach → chronologies, RHA 1988 « L'aéronavale » pour l'histoire. ≈ 8-12 fiches.

Priorité 4 — appareils d'entraînement transverses (fiches gabarit appareil) :
PC-21, Grob 120, Cirrus SR20/22, Alphajet, TB-30 Epsilon (retiré 2019), Fouga Magister
(historique), Calliopé/NH90 école. ≈ 5-7 fiches.

Priorité 5 — Fondamentaux/facteurs humains : études de cas sécurité depuis rapports BEA-É
(via Aerobuzz), réglementation, radio. ≈ 4-6 fiches. Psychotechnique : vision spatiale,
mémoire, attention. ≈ 3 fiches.

Réserves futures (corpus dédiés) : COMAERO Formation t.1-3, Hélicoptères, Avions militaires ;
Persée RHA numéros thématiques. Chacun = campagne d'ingestion type « Guide EOPAN ».
